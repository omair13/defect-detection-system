## 📋 What is defect-detection-system?

This application helps you find defects in products automatically. It uses smart image analysis to check the quality of items as they pass by. The system shows results quickly, with high accuracy. You do not need to be a technical expert to use it.

This tool uses Python and OpenCV. It was built with an easy interface to work on Windows computers. You can use it to improve quality control in manufacturing or inspection tasks.

## ⚙️ Behind the Scenes

defect-detection-system uses OpenCV for image processing. It applies machine learning models for spotting defects automatically. The application runs locally, so your images stay private.

It communicates using a REST API built with FastAPI, but you do not need to configure anything to use the app on your Windows computer.
